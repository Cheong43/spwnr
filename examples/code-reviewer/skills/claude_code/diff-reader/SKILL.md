# diff-reader

Reads git diffs using Claude Code specific bindings.

## Tool Binding Notes

Prefer the diff and repository inspection bindings that exist in Claude Code. Do not reference Codex, Copilot, or OpenCode tool names from this file.

## Capabilities

- Review staged and unstaged diffs in Claude Code
- Follow file references from a diff into relevant implementation files
- Keep findings aligned with the tools available in Claude Code sessions
